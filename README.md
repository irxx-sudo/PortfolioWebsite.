# Portfolio Website

This repository contains my **first personal portfolio website**, created to showcase my skills, projects, and learning journey in web development.

This portfolio website is built using **HTML, CSS, and JavaScript**.  
The main goal of this project is to practice front-end development fundamentals and understand how a complete website is designed, structured, and deployed.

It reflects my initial step into web development and will be continuously improved as I gain more experience.
